# Final Project Overview

##Part 1 Requirements

1. Created "movies" laravel project
2. Used Rotten Tomatoes REST service to gather information regarding box office movies and top films out.
3. Utilized Google Maps to show recent tweets related to the movie from around the world.
4. Utilized AJAX to search for movies from Rotten Tomatoes and bring up clickable results which then lead to more detailed information.
5. jQuery functoins and interactivity involved in many of the processes.

Other: Utilized google trends to show trends related to movie clicked.

To meet second half requirements:
1. Data validation using Laravel's Validator class to validate account creation process & forms
2. Client-side templating with Handlebars to generate twitter feeds.

Additional:
Blade tempating used in combination with the validator to show whether account creation was successful or not.
